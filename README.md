# Hey, I'm Iliyan 👋

CS student at University College Cork, originally from Bulgaria. I like building things for the web and occasionally mess around with data science and hardware projects.

Mostly working with **TypeScript**, **React/Next.js**, **C#/.NET**, and **Python** these days.

### Some things I've built

- [**WatchHeaven**](https://github.com/iliyanCoding/watch-heaven) — a watch marketplace built with ASP.NET Core, SignalR for real-time updates, and SQL Server
- [**NBAdle**](https://github.com/iliyanCoding/nbadle) — an NBA guessing game inspired by Wordle, built with React
- [**SIR Model**](https://github.com/iliyanCoding/sir-model) — simulating disease spread with Python and SciPy
- [**Basketball Scouting Platform**](https://github.com/iliyanCoding/basketball-scouting-platform) - a basketball scouting tool built with Flask and Bootstrap
